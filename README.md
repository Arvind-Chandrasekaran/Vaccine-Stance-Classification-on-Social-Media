# Vaccine Stance Classification on Social Media

### Overview
This repository contains the resources used to develop and evaluate a language-model-based system for classifying vaccine-related stances expressed on social media.  
For methodological details and results, refer to **Report.pdf**.

---

### Repository Structure

- **src/**  
  Code used for model training, evaluation, and preprocessing.

- **input_data/**  
  Training and test datasets used in the study.

- **prediction/**  
  Model-generated predictions on the provided test data.

---

### Temporal Analysis
Code, results and data from the temporal analysis conducted in the study are not included in the public repository.
